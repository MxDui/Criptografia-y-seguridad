# Código Fuente - Tarea 1 de Criptología

Este repositorio contiene los programas solicitados para la Tarea 1.

## Archivos Principales

Los problemas que explícitamente pedían código son:

*   `problema_1_2_cesar_polibio.py`: Contiene las soluciones programadas para el **Problema 1 (Cifrado César)** y el **Problema 2 (Cuadrado de Polibio)**.
*   `problema_3_vigenere_kasiski.py`: Contiene el análisis de **Kasiski** para el **Problema 3**.

## Otros Scripts Elaborados (Análisis y Pruebas a Mano)

Además, se incluyen otros scripts que fueron utilizados para verificar y automatizar cálculos para los problemas que debían resolverse "a mano":

*   `problema_4_cesar_fuerzabruta.py`: Pruebas de fuerza bruta para el Problema 4 (César).
*   `problema_5_7_afin_hill_math.py`: Operaciones en limpio de álgebra matricial modular para los Problemas 5 y 7.
*   `problema_6_vigenere.py`: Cifrado y descifrado de Vigenère para verificar los incisos del Problema 6.
